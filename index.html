Actuellement en formation chez Simplon.co, nous sommes à la recherche d'un stage non rémunéré de 2 mois du 22 mai au 18 juillet prochain. Découvrez nos profils.

L'école Simplon

Simplon est une école de formation professionnelle en numérique, axée sur la diversité et l'inclusion. Elle propose des formations intensives et gratuites pour former des développeurs web et mobiles qualifiés, prêts à intégrer le marché du travail. Simplon a pour mission de démocratiser les compétences numériques pour tous.

Compétences visées et Stack technique

8 compétences sont attendues afin de valider notre diplôme :

  • Créer une base de données
  • Développer les composants d’accès aux données
  • Élaborer et mettre en œuvre des composants dans une application de gestion de contenu ou e-commerce
  • Maquetter une application
  • Réaliser une interface utilisateur web
  • Développer une interface utilisateur web dynamique
  • Réaliser une interface utilisateur avec une solution de gestion de contenu ou e-commerce
  • Développer la partie back-end d'une application web ou web mobile
